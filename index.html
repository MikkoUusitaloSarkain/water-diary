Supabase setup

Add your Supabase project URL and public anon key. They are saved only in this browser.

Create this table in Supabase SQL Editor first:

create table if not exists water_entries ( id uuid primary key default gen_random_uuid(), entry_date date not null, entry_time time not null, amount_ml integer not null check (amount_ml > 0), drink_type text not null default 'water', note text, created_at timestamptz not null default now() ); alter table water_entries enable row level security; -- Personal/simple setup: allow public anon access to this table. -- For a private multi-device app, add Supabase Auth and user_id policies instead. create policy "Allow anon read" on water_entries for select using (true); create policy "Allow anon insert" on water_entries for insert with check (true); create policy "Allow anon delete" on water_entries for delete using (true);

Note: the above anon policy is simple but not private. For a private app, use Supabase Auth before sharing the URL publicly.
